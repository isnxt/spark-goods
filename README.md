# spark-goods
spark流式计算    
- 电商商品关注度
- 推荐系统
- 关联系统

## core
anaylse (spark-goods)    
- spark:2.6.5
- hadoop:2.6.0
- scala:2.11.8

website (springboot1)    
- boostrap
- startbootstrap
- echarts
- springboot
- mysql

## step
1. run Server.java + Client.java
2. run trainEvaluation.scala
3. run trainAIS.scala
4. run fpg.scala
5. run springboot project 
6. open http://localhost:8080/ in Web Browser
